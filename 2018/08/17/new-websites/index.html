「公告」新世界～

发表于 2018-08-17 | 分类于 泛 - 公告 | 阅读次数:
字数统计: 358

我的网站又迁移了。。。这次变化巨大！

改变

域名

一级域名依然是 pazyx.xyz ，但这次我弄了好几个站点～～

  • pazyx.xyz 主页
  • blog.pazyx.xyz 博客
  • art.pazyx.xyz 艺术馆（在建）
  • download.pazyx.xyz 文件服务
  • …

服务器

博客由原来托管在 GitHub Pages 上到运行在 Apache 服务器上。
在 Google PageSpeed 优化后访问速度较之前提高了不少。

所有站点都运行在一台 VPS 上，管理方便很多。为了查看 git commit 方便博客也同时运行在 GitHub Pages 上。

https

全站开启 https，禁用 IP 地址访问和 80 端口访问。保护了网页在一定程度上不受 GFW 影响，也避免了运营商篡改网页。

增加

pazyx.xyz

我的个人主页，作为各个站点的总导航。

art.pazyx.xyz

我的艺术馆，我会将我的作品展示在上面。

download.pazyx.xyz

我的一些文件，便于我使用并为其他站点提供文件下载支持。

后续

10print.pazyx.xyz

编写一个 Javascript 的 10print 程序运行在 10print.pazyx.xyz 上。

test.pazyx.xyz

我的项目测试，之前我的 hexo 主题测试就放在上面，之后会有新的东西挂在上面。

博客清理

我希望我的博客可以更干净一点，比如 bookmark、VobBw、seesee 这些页面完全可以放在 pazyx.xyz 上。之后会做一些这方面的改变。
